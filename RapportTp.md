# TP 3 – Formulaires Symfony

## Description
Ce TP a pour objectif de transformer une page HTML statique en un formulaire dynamique en utilisant le framework Symfony et le moteur de templates Twig.

## Travail réalisé
-Création du formulaire Symfony (ProductFormType)
Un formulaire personnalisé a été créé afin de définir les champs nécessaires à la sélection du produit.

-Utilisation des champs IntegerType et ChoiceType
Les types de champs Symfony ont été utilisés pour gérer la quantité du produit et le choix de la couleur.

-Traitement du formulaire dans le contrôleur
Le formulaire est géré dans le contrôleur pour traiter la soumission et récupérer les données saisies.

-Affichage et personnalisation avec Twig
Le formulaire est affiché avec Twig en utilisant les fonctions Symfony dédiées à la génération des formulaires.

-Intégration de Bootstrap 5.3
Bootstrap a été intégré afin d’assurer une mise en page responsive et une interface utilisateur claire.
